# 项目设置

## 快速开始

| 场景 | 操作 |
|------|------|
| **首次调试** | 告知项目目录 → AI 执行 `--init` → 采集参数 → 配置生成在项目目录 |
| **恢复调试** | AI 自动查找项目目录下的 `embedded-debug-config.json`，直接进入调试循环 |
| **查看帮助** | 输入 `debug help` 查看详细规范 |

## 配置存放位置

配置文件 `embedded-debug-config.json` 存放在**项目根目录**，便于团队共享和版本控制（uv4_path 除外，它是固定系统路径）。
