# 调试检查清单

## 每次调试迭代完成后检查

- [ ] 串口参数（端口号、波特率）是否正确？
- [ ] 编译是否 0 Error 0 Warning？
- [ ] 下载是否成功（`Flash Load finished`）？
- [ ] 串口日志中是否有预期的调试打印？
- [ ] 调试打印信息是否足够定位问题？
- [ ] 修改是否只影响目标模块？
- [ ] 修复后是否移除了调试代码？
- [ ] 最终编译下载是否通过？

## 常见问题速查

| 问题 | 原因 | 解决 |
|------|------|------|
| `Access to COM port is denied` | 串口被其他程序占用 | 关闭串口工具 |
| 串口收到乱码 | 波特率不匹配 | 确认目标板实际波特率 |
| 编译未生效 | 文件未被重新编译 | 删除 `.o` 文件或 `-c` 清理 |
| `#ifdef` 打印未出现 | `CHESHI` 宏未定义 | Keil 工程添加 `CHESHI` |
